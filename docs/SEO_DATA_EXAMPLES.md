# SEO data examples

Local reference for slugs, alt text, and meta fields used across the admin panel.

---

## Slider (hero images)

| Field   | Example value                    | Notes                          |
|---------|----------------------------------|--------------------------------|
| Slug    | `summer-sale-hero`               | Lowercase, hyphens only        |
| Alt     | `Woman applying natural face oil at spa` | Short, descriptive     |
| Title   | Summer Sale – 20% Off             | Shown on slide                 |
| Href    | /shop?promo=summer               | Link target                    |

**Alt guidelines:** Describe the image in one line; avoid “image of” or “picture of”.

---

## Product images

| Field   | Example value                          | Notes                    |
|---------|----------------------------------------|--------------------------|
| Slug    | `rose-hip-oil-50ml`                    | For filename on upload   |
| Alt     | `Rose hip oil 50ml bottle, Nature Secret` | Product + variant     |

**Per-image alt:** Match order with image (first alt = first image). Use product name + variant or angle.

---

## Blog featured image

| Field        | Example value                                      | Notes              |
|--------------|----------------------------------------------------|--------------------|
| Slug         | `lavender-skin-benefits`                           | For upload filename|
| Image alt    | `Lavender flowers close-up for skincare article`   | Describes image    |
| SEO title    | `Lavender for Skin: Benefits and Uses \| Nature Secret` | 50–60 chars  |
| SEO description | `Learn how lavender soothes and heals skin. Tips and product picks.` | 150–160 chars |

**SEO title:** Include brand; keep under ~60 characters.  
**SEO description:** One clear sentence; 150–160 characters.

---

## Product page (general)

- **Slug:** `rose-hip-facial-oil` (from product name, hyphenated).
- **Name:** Full product name as shown (used in meta when no dedicated SEO fields).

---

## Quick checklist

- [ ] Slider: slug + alt on upload; title and href set.
- [ ] Products: slug + alt per image; imageAlts in order.
- [ ] Blog: featured image alt; slug on upload; SEO title and description filled.
- [ ] All slugs: lowercase, a–z, 0–9, hyphens only; no spaces or special chars.
