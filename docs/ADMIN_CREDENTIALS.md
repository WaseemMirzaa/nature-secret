# Admin & staff login

Use these after running the backend seed (`cd backend && npm run seed`).

| Role   | Email                     | Password   |
|--------|---------------------------|------------|
| Admin  | `admin@naturesecret.pk`  | `Admin123!` |
| Staff  | `staff@naturesecret.pk`  | `Staff123!` |

**Admin panel:** `/admin` (or `/admin/login` when not logged in).

Staff can update order delivery status; admin has full access.
